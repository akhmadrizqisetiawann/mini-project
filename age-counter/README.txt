# Age Calculator

This Python script calculates the number of days since your birth and your age in years, months, and days. It uses object-oriented programming (OOP) principles for a clean and modular approach.

## Features

- Input validation for date of birth
- Calculation of the number of days since birth
- Calculation of age in years, months, and days

## Requirements

- Python 3.x

## Installation

To use this script, ensure you have Python installed on your system. You can download it from the [official Python website](https://www.python.org/downloads/).

No additional libraries are required.
