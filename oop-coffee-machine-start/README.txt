# Coffee Machine Program

This program simulates a coffee machine that allows users to select and purchase different types of coffee. The program handles resource management, payment processing, and keeps track of profits.

## Features

- **Menu Management**: Displays available coffee options.
- **Resource Management**: Tracks the available resources (water, milk, coffee).
- **Payment Processing**: Handles coin input and calculates if sufficient funds have been inserted.
- **Reporting**: Provides reports on resources and profits.
- **User Interaction**: Allows users to interact with the coffee machine through simple text inputs.

## Classes

### CoffeeMaker
Models the machine that makes the coffee. It manages the resources needed to make each type of coffee.

### MenuItem
Models each menu item, holding the necessary ingredients and cost.

### Menu
Manages the menu, holding the list of available drinks.

### MoneyMachine
Handles the payment processing, including coin input and calculating change.
